# StayBliss

A hotel booking frontend built with plain HTML, CSS and JavaScript.

## Pages

- **index.html** — Lists all hotels with search and sort
- **hotel.html** — Hotel details with photo gallery
- **booking.html** — Booking form with live price summary

## API

`https://demohotelsapi.pythonanywhere.com/hotels/`

## How to run

Open in VS Code, right click `index.html`, click "Open with Live Server".
